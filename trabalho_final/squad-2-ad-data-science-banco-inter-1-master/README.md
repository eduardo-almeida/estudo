1. Instale o virtualenv e crie o ambiente virtual
    * `pip install virtualenv`
    * `virtualenv projetoFinal`

2. Ative o ambiente virtual criado
    * `cd projetoFinal`
    * `source bin/activate`

	* `git init`

4. Instale as dependencias do projeto
	* `pip install -r requirements.txt`
